# Resend POC

Quick start for the frontend and backend.

## Backend (Deno + Hono)

```bash
cd backend
export RESEND_API_KEY=your_key
deno task dev
```

## Frontend (Vite + React)

```bash
cd frontend
npm install
npm run dev
```

## Notes

- Backend defaults to `http://localhost:8000`.
- Frontend defaults to `http://localhost:5173`.
